# Online Word Counter

字数统计器专题项目，用于统计中文字数、英文单词、字符数、行数、段落数和阅读时间。

## Online Tool

[字数统计器](https://www.abplus.xin/tools/word-counter)

## Features

- 中文字数统计
- 英文单词统计
- 字符数和行数统计
- 段落数和阅读时间估算

## Use Cases

- 检查文章字数
- 统计文案长度
- 控制产品说明字数
- 估算文档阅读时间
